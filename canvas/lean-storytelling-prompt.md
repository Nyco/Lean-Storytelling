# Lean Storytelling Canvas with LLM Prompt

## How to Use This Canvas

This canvas helps you structure your story using the Lean Storytelling methodology and then generate a polished narrative with an LLM. Follow these steps:

1. **Copy this file** to create your own version (e.g., `my-story.md`)
2. **Fill in each section** with your specific information (DO NOT modify the prompt section)
3. **Be concise and specific** - focus on the most important details
4. **Copy the entire file** and paste it into your LLM interface
5. **The LLM will read the prompt** and generate your story automatically

## LLM PROMPT (DO NOT MODIFY)

**Instructions for the LLM:**

Read the entire file and extract the content from all sections above (Context, Target, Empathy, Problem, Consequences, Solution, Benefits, Why). Then generate a professional, well-structured story that follows the Lean Storytelling framework.

**Requirements:**
- Use all the information provided in the sections above
- Structure the story in the exact order: Context, Target, Empathy, Problem, Consequences, Solution, Benefits, Why
- Write in a neutral and professional tone
- Use clear, concise sentences
- Remove any ambiguity or vague language
- Ensure smooth transitions between sections
- Keep the story focused and impactful
- Do not add any information not provided in the sections above
- Do not include any explanations, analysis, or additional commentary
- Output only the final story text

**Output Format:**
Generate the story as a continuous narrative text, with each Lean Storytelling element flowing naturally into the next. Do not label the sections or include any formatting instructions in your response.
